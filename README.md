# Dashboard Marketing - Online Retail II

**Projet ECE Data Visualization 2025**

Ce projet est une application d'aide à la décision pour les équipes marketing. Elle permet d'analyser les transactions e-commerce, de mesurer la rétention client par cohortes, de segmenter la base via la méthode RFM et de simuler la valeur vie client (CLV).

---

## Fonctionnalités

L'application est divisée en 4 modules principaux :

1.  **Overview (KPIs)** : Vue d'ensemble instantanée de la santé du business (CA, Clients Actifs, Rétention globale, Part Acquisition vs Rétention).
2.  **Diagnostic (Cohortes)** : Analyse fine de la rétention via Heatmap et courbes de densité de valeur par âge (pour repérer les décrochages).
3.  **Priorisation (Segments RFM)** : Segmentation de la base client (Champions, Fidèles, À risque...) pour cibler les actions marketing.
4.  **Simulation (Scénarios CLV)** : Outil prédictif permettant de tester l'impact d'une hausse de rétention ou de marge sur la Valeur Vie Client.
